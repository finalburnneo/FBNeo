Pre-Building:
1) download and run visual studio 2015 IDE community edition installer from https://visualstudio.microsoft.com/ 
2) select Desktop development with C++
3) next you will need a perl interpreter for windows. I recommend Compact Portable Perl for Windows (https://www.southsoftware.com/perl.html). select the version compatible with your operating system.
4) Place the downloaded perl.exe into the root directory of the project.
5) If you have MSYS2 installed with the corresponding Perl toolchain, you can skip steps 3 to 4 above and add [installation path]\msys64\usr\bin\ to your system PATH environment variable.

Building
1) checkout the code using git (Optionally: fork it first!)
2) in the checked out code, open projectfiles then visualstudio-2015, then open fba_vs2015.sln
3) select the correctly platform and build type. 
4) hit build!
