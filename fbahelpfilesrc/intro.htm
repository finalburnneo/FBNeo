FB Neo

Introducing FB Neo

FB Neo is an arcade emulator supporting the following platforms;

  • Capcom CPS-1
  • Capcom CPS-2
  • Capcom CPS-3
  • Cave
  • Data East DEC-0, DEC-8, Cassette System, DECO IC16 and DECO-32 based games
  • Galaxian based hardware
  • Irem M62, M63, M72, M90, M92 and M107 hardware
  • Jaleco MegaSystem 32
  • Kaneko 16
  • Konami
  • Namco Mappy, System 86, System 1 & 2, NA-1, NA-2 and others
  • Neo-Geo
  • NMK16
  • Pacman based hardware
  • PGM
  • Psikyo 68EC020 and SH-2 based hardware
  • Sega System 1, System 16 (and similar), System 18, System 32, X-Board and Y-Board
  • Seta-Sammy-Visco (SSV) based hardware
  • Super Kaneko Nova System
  • Toaplan 1
  • Toaplan 2
  • Taito F2, F3, X, Z and others
  • Miscellaneous drivers for lots of other hardware

FB Neo also has console/computer drivers for Sega Megadrive/Genesis, ColecoVision, Sega SG-1000, Sega Master System/Game Gear, MSX-1, ZX Spectrum, PC-Engine/TurboGrafx 16/SuperGrafx, Neo-Geo Pocket. and also Fairchild Channel F

FB Neo can currently be obtained from https://github.com/finalburnneo/FBNeo/releases/tag/latest.
